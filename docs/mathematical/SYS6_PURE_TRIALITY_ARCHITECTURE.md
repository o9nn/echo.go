## 2. Triad of Dual-State Quanta

### 2.1. Structure

**Definition**: 3 qubits, each of order 2 (pairwise entanglement)

**State space**:
```
|Ψ⟩ = |E₁⟩ ⊗ |E₂⟩ ⊗ |E₃⟩
```

Where each entanglement is:
```
|E₁⟩ = α₁|00⟩ + β₁|01⟩ + γ₁|10⟩ + δ₁|11⟩  (4 states)
|E₂⟩ = α₂|00⟩ + β₂|01⟩ + γ₂|10⟩ + δ₂|11⟩  (4 states)
|E₃⟩ = α₃|00⟩ + β₃|01⟩ + γ₃|10⟩ + δ₃|11⟩  (4 states)
```

**Total dimension**: 4 × 4 × 4 = **64 states**

### 2.2. The 3 Pairwise Entanglements

**Entanglement 1**: Threads (P1, P2)
```
|E₁⟩ = Q₁ ⊗ Q₂
```

**Entanglement 2**: Threads (P3, P4)
```
|E₂⟩ = Q₃ ⊗ Q₄
```

**Entanglement 3**: Threads (P5, P5') (meta-thread)
```
|E₃⟩ = Q₅ ⊗ Q₅'
```

**Interpretation**: 
- E₁ and E₂ are **spatial** entanglements (different thread pairs)
- E₃ is a **temporal** entanglement (same thread at different times)

### 2.3. Geometric Representation

The 3 entanglements form a **triangular prism**:

```
      E₃
     /  \
    /    \
   E₁----E₂
```

Each edge represents a pairwise entanglement. The prism has:
- **3 edges** (the 3 entanglements)
- **2 triangular faces** (top and bottom)
- **3 rectangular faces** (sides)

**Total**: 3 + 2 + 3 = 8 elements (corresponding to 2³)

## 3. Dyad of Tri-State Quanta

### 3.1. Structure

**Definition**: 2 qubits, each of order 3 (triadic entanglement)

**State space**:
```
|Ψ⟩ = |T₁⟩ ⊗ |T₂⟩
```

Where each triad is:
```
|T₁⟩ = Σᵢⱼₖ αᵢⱼₖ |i⟩ ⊗ |j⟩ ⊗ |k⟩  (8 states)
|T₂⟩ = Σᵢⱼₖ βᵢⱼₖ |i⟩ ⊗ |j⟩ ⊗ |k⟩  (8 states)
```

**Total dimension**: 8 × 8 = **64 states**

### 3.2. The 2 Triadic Entanglements

**Triad 1**: Threads (P1, P2, P3)
```
|T₁⟩ = Q₁ ⊗ Q₂ ⊗ Q₃
```

**Triad 2**: Threads (P2, P3, P4) (overlapping)
```
|T₂⟩ = Q₂ ⊗ Q₃ ⊗ Q₄
```

**Key Property**: The two triads **share threads** P2 and P3, creating a **nested overlap**.

### 3.3. Geometric Representation

The 2 triads form a **double tetrahedron** (stella octangula):

```
       P1
      /|\
     / | \
    /  |  \
   P2--P3--P4
    \  |  /
     \ | /
      \|/
       P5
```

Each tetrahedron has:
- **4 vertices** (threads)
- **6 edges** (pairwise connections)
- **4 faces** (triadic surfaces)

**Overlap**: P2 and P3 are shared between the two tetrahedra.

**Total**: 2 tetrahedra = 2 × 4 faces = 8 faces (corresponding to 2³)
